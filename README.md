# Adult Census Income Prediction

## Description
This project predicts whether a person's income exceeds $50K/year
using the Adult Census Income dataset.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Workflow
- Data cleaning
- Exploratory Data Analysis
- Feature encoding
- Model training
- Model evaluation

## Result
The model classifies income into:
- <=50K
- >50K
